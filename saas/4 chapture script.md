SAAS Script

Variables
It represents the data such as numeric values, characters or memory addresses.
```
$txtcolor:#008000;
$fontSize: 20px;

p{
   color:$txtcolor;
   font-size:$fontSize;
}
```

## Operations
### Number Operations
```
$size: 25px;

h2{
   font-size: $size + 5;
}

h3{
   font-size: $size / 5;
}
```
